About this tool

This tool uses DuckDB-WASM to load your data entirely in the browser — nothing is uploaded to any server.

Once loaded, select coordinate columns and choose which KYTC roadway attributes to fetch from the Kentucky Transportation Cabinet Spatial API. The API returns LRS (Linear Referencing System) attributes such as county, route, milepoint, speed limit, and more for the nearest roadway to each GPS coordinate.

When processing is complete, download the enriched dataset in your preferred format.
