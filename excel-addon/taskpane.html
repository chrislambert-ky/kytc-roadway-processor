KYTC Roadway Processor

Batch processing and custom formulas for the Kentucky Transportation Cabinet Linear Referencing System (LRS) Spatial API.

What you can do
  • Process Sheet — writes LRS attributes directly into your workbook as static values for every coordinate row. Re-run weekly to stay current with the LRS.
  • Field Reference — browse and copy all available return field names, filtered to 100%-reliable fields by default.
  • Clear cache — forces fresh API calls on the next run or formula recalc.
Custom formulas
  • =KYTC.LRS(lat, lon, "Field") — single field lookup
  • =KYTC.LRS_MULTI(lat, lon, "Field1,Field2,…") — spills right
  • =KYTC.FIELDS() — spills the full field catalog with coverage info
